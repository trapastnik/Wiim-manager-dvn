# Установка и запуск WiiM Web Control

## 1. Установка Node.js

### macOS
```bash
# Установка через Homebrew
brew install node

# Или скачайте установщик с https://nodejs.org
```

### Проверка установки
```bash
node --version
npm --version
```

## 2. Установка зависимостей проекта
```bash
cd /Users/dvn/Desktop/ВВВВВВВВ/wiim
npm install
```

## 3. Настройка

Файл `.env` уже создан с адресом вашего WiiM:
```
WIIM_IP=192.168.1.156
PORT=3000
WIIM_USE_HTTPS=true
```

## 4. Запуск

```bash
npm start
```

Откройте браузер: http://localhost:3000

## 5. Сканирование сети (опционально)

Если IP-адрес плеера изменился, используйте сканер:

```bash
node scan.js
```

Он найдет все WiiM устройства в сети 192.168.1.0/24
